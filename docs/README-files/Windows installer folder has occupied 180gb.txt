1

Where is that installer folder is that C:\Windows\Installer?

If so, it is not safe to manually clean that folder yourself, but you can use a free utility named PatchCleaner to safely clear out that folder, it is quite safe to do so, that utility as developers specifically to do that task:

https://sourceforge.net/projects/patchcleaner/

2

To see if there is any more space on your drive that can be cleaned:

Click the link below to download a small free utility called Treesize. Using that tool, you will instantly see where this space is being used

https://www.jam-software.com/treesize_free/

To run TreeSize, right click its icon and choose Run as Administrator.

Please provide a screenshot of your C drive in that TreeSize utility, so I can see your hard drive and work out where that space is being used and if there is an underlying problem.

hi there abbi_123

If you mean it is the Windows Local Disk C, then It is not recommended to delete the files, including the "tmp" files. You may look for a file called Windows.old or some similar names. You can find Windows.old or other big files by checking each folder's size one by one in the installer disk to see which one is tacking up all the space. Though it is still not recommended to delete them yourself. Instead, you can run cleaning programs like CCleaner to let it decide and remove backup and tempt files.

You can also look here for more information: Cleaning the Windows\Installer folder - Microsoft Community
