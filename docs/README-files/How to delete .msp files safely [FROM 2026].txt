Open | Software

It depends which directory they're located in. If they're in the windows\softwaredistribution\download you can safely delete that whole directory. Same with temp folders. But many others you do not want to touch.

When running disk cleanup, make sure you click the "clean up system files" option as that will find more stuff.

But in reality, if you're desperate for 30 gigs, your drive is too small. Assuming it is an SSD, you want it below 90% usage all the time, and ideally below 80. If you're that short on space, this will be a constant battle and it is harming your performance.

Also if your system is stable cleanup Microsoft update history by running the following in order inside an elevated command prompt (run as admin)

DISM.exe /online /Cleanup-Image /StartComponentCleanup
Followed by:

DISM.exe /online /Cleanup-Image /StartComponentCleanup /ResetBase
This removes your ability to uninstall Windows updates, but can save many GBs of storage by not retaining old stuff.
